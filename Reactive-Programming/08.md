# 08. Backpressure
- 배압 또는 역압
- 흐름을 제어하기 위해 가해지는 압력
- Reactor에서 Backpressure 처리 방식
  - 데이터 개수 제어
  - 전략
    - IGNORE : Backpressure 적용 안함
    - ERROR : 버퍼 찼을때 예외 발생
    - DROP : 버퍼 밖에서 대기하는 데이터부터 drop
    - LATEST : 버퍼 밖에서 대기하는 가장 최근 데이터부터 버퍼에 채우는 전략
    - BUFFER : 버퍼 안에 있는 데이터부터 drop
      - 폐기
      - drop
      - 버퍼링
